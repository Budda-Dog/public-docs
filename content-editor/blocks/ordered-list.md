# Ordered lists

Ordered or numbered lists help you prioritize items or create a list of steps.

### Example of ordered list

1. Item 1
   1. Nested item 1.1
      1. Nested item 1.1.1
   2. Nested item 1.2
2. Item 2
3. Item 3

{% hint style="info" %}
To create nested items, you can use `tab` to indent and `shift` + `tab` to outdent.
{% endhint %}

### Representation in Markdown

```markdown
1. Item 1
   1. Nested item 1.1
      1. Nested item 1.1.1
   2. Nested item 1.2
2. Item 2
3. Item 3
```
